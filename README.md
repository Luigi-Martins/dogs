## Pré-requisitos

Antes de começar, você precisa ter instalado:

- Node.js (recomendado versão 18+)
- npm (ou yarn/pnpm)

Verificar versão:

```bash
node -v
npm -v
```

---

## Rodando local (desenvolvimento)

### 1. Instalar dependências

```bash
npm install
```

### 2. Rodar o projeto

```bash
npm run dev
```

### 3. Acessar no navegador

```
http://localhost:5173/
```

---

## Build do projeto

Gerar build de produção:

```bash
npm run build
```

Visualizar build:

```bash
npm run preview
```

---

## Lint e padronização

Rodar lint:

```bash
npm run lint
```

Corrigir automaticamente:

```bash
npm run lint --fix
```

## Plugins e configurações

O projeto já vem configurado com:

- ✔️ ESLint configurado
- ✔️ React Router DOM para navegação
- ✔️ PropTypes para tipagem de props
- ✔️ vite-plugin-svgr (uso de SVG como componente React)
- ✔️ Estrutura inicial com Vite
